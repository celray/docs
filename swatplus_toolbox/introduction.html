  • format_align_centerIntroduction

    SWAT+ is relatively new and there are not so many tools to carry out model analysis and calibration. SWAT+ Toobox is a free tool that allows the user to perform. analyses, calibration and more.

    The software has been writen in C# and is currently still in testing. Users interested to test it can try it by downloading an installer here.

  • codeSetting up software

    After downloading the installer, install normally using the default settings. Note that you will have to uninstall any previous versions before installing a new version.
