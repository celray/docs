  • format_align_centerCreating and Opening a Project

    Open SWAT+ Toolbox by clicking on the shortcut from start menu or the Desktop. To create a project, click on New Project button in the Home page. A dialog box will appear. This dialog box allows to select a directory. Brows to the TxtInOut directory and click "OK". Another Dialog box will appear and prompt you to save your SWAT+ Toolbox Project

    Figure 2: The home page of SWAT+ Toolbox software

    You can open a project by clicking the "Open Project" button and selecting an existing project.
