Create a new project

There are two ways to set up a new project.

  • Using QSWAT+ Integration
  • Using the home page of SWAT+ Toolbox

1. Using QSWAT+ Integration

If you have set up QSWAT+ Integration, you will have the option, 'SWAT+ Toolbox'. Click on it to create a new SWAT+ Toolbox. This creates a project with same name as your QSWAT+ project and in the same directory as the QSWAT+ project.

You will see a prompt notifying you details about a new SWAT+ Toolbox project.

2. Using the Home Page of SWAT+ Toolbox

On the homepage of SWAT+ Toolbox, Click on New Project, then Browse to your TxtInOut directory to select it.

Once a valid TxtInOut directory is selected, you will be prompted to save a SWAT+ Toolbox Project file (.spt) browse to where you want to save it and save.
