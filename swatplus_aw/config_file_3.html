    Management Options

    SWAT+ AW allows you to specify land use management options and reservoir management options. To do this fill the management sections in the config file as shown in Figure 4. To set the management for more than one land use, repeat the block with curly brackets with different landuse name and options inside them. Refer to the SWAT+ documentation for more details on land use types in SWAT+
